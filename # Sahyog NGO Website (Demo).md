# Sahyog NGO Website (Demo)

Simple, clean NGO website for Sahyog built using HTML, CSS, and JavaScript.

## Pages
- Home
- About
- Our Work
- Get Involved
- Donate
- Contact

## How to run
Open `index.html` in a browser.

## Deployment
- For demo: host using GitHub Pages
- For final: upload all files to Hostinger `public_html`
